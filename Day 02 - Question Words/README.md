## How to Form Questions in German

Alright, so German has two types of questions. Let me make sure I understand both:

### A) Ja/Nein Fragen (Yes/No Questions)

**The structure is:** Verb → Subject → Rest

This is different from English! In English, we use "Do" or "Is" at the beginning, but in German, the verb just comes first.

**Example:**
```
Kommst du aus Iran?
Are you from Iran?
```
*(فعل میاد اول جمله. مثل انگلیسی نیست که Do بیاد.)*

**More examples:**
- `Hast du Zeit?` → Do you have time?
- `Ist das dein Buch?` → Is this your book?

### B) W-Fragen (Wh-Questions)

**The structure is:** W-word → Verb → Subject → Rest

**Example:**
```
Wo wohnst du?
Where do you live?
```
*(کلمه سوالی اول میاد، بعد فعل، بعد فاعل.)*

**Another example:**
```
Wann beginnt der Kurs?
When does the course start?
```

## My Simple Method: How to Choose the Right W-Frage

I've come up with a simple method so I always know which W-Fragewort to use. Here's my thinking process:

### Step 1: Ask Myself "What Information is Missing?"

Every W-Fragewort corresponds to a type of missing information. Let me map this out:

| **If I want to know…**     | **I use this W-Frage…**           | **Persian Explanation** |
| ---------------------------- | ------------------------------- | ----------------------- |
| a **person**                 | **Wer?**                        | چه کسی؟                 |
| an **object / activity**     | **Was?**                        | چه چیزی؟                |
| a **place (location)**       | **Wo?**                         | کجا؟                    |
| **destination**              | **Wohin?**                      | به کجا؟ (جهت)           |
| **origin / from where**      | **Woher?**                      | از کجا؟                 |
| **time**                     | **Wann?**                       | کی؟                     |
| a **reason**                 | **Warum? / Wieso?**             | چرا؟                    |
| **manner / method**          | **Wie?**                        | چگونه؟                  |
| **quantity (uncountable)**   | **Wieviel?**                    | چقدر؟                   |
| **quantity (countable)**     | **Wie viele?**                  | چند تا؟                 |
| **choosing between options** | **Welcher / Welche / Welches?** | کدام؟                   |

### Step 2: Remove One Part from the Sentence

I look at the original sentence, remove one part, and that missing part tells me which question word to use.

**Example:**

Original sentence:
```
Du wohnst in Berlin.
```

If I remove: `in Berlin`  
Missing information = place  
→ So I use **Wo?**

**Rule:** The question word replaces the missing info.

**Result:**
```
Wo wohnst du?
```

### Step 4: The Question Type Guide (Super Simple)

When I'm stuck, I just think about what type of answer I'm looking for:

**If the answer is a PERSON → Wer?**
```
Er spricht mit Ali. → Mit wem spricht er?
```

**If the answer is a THING → Was?**
```
Sie trinkt Kaffee. → Was trinkt sie?
```

**If the answer is a PLACE (no movement) → Wo?**
```
Ich arbeite im Büro. → Wo arbeitest du?
```

**If the answer is movement TO somewhere → Wohin?**
```
Wir fahren nach Deutschland. → Wohin fahrt ihr?
```

**If the answer is movement FROM somewhere → Woher?**
```
Ich komme aus Iran. → Woher kommst du?
```

**If the answer is TIME → Wann?**
```
Der Kurs beginnt um 9 Uhr. → Wann beginnt der Kurs?
```

**If the answer is REASON → Warum?**
```
Ich lerne Deutsch, weil es wichtig ist. → Warum lernst du Deutsch?
```

## Quick Summary for Myself

- German has two question types: Ja/Nein Fragen and W-Fragen
- W-Fragen structure: W-word → Verb → Subject → Rest
- I can use the Persian shortcut to remember most of them
- When in doubt, think: "What information am I missing?"
- The question word replaces the missing information in the sentence
